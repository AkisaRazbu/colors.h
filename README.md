### Colors.h
A basic C library that adds color to your output

### Installation
```
git clone https://github.com/AkisaRazbu/colors.h && cd colors.h
```

```make
sudo make clean install
```

###	Usage
```c
#include <colors.h>

int main(){
	red();
	printf("Color.h project");
	reset();
	return 0;
}
```
